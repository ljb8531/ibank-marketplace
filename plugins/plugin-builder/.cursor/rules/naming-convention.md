---
name: naming-convention
description: name 필드 kebab-case, 폴더명과 name 일치, 구체적 키워드
alwaysApply: true
---

모든 name 필드는 lowercase kebab-case만 허용한다. 스킬 폴더명과 SKILL.md 내 name 프론트매터는 반드시 동일해야 한다. 에이전트 파일명이 곧 에이전트 name이다. 커맨드 파일명이 슬래시 커맨드명이 된다. 프론트매터의 description에는 에이전트 라우팅에 사용될 구체적 키워드를 포함한다. 추상적 표현을 금지한다.

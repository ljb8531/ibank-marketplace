모든 Skill md와 Agent md는 500자 미만으로 작성한다. 초과 시 반드시 세분화하여 복수 파일로 분리한다. 세분화 기준은 T-T-D-A-V 단계별, 의사결정 분기별, 또는 도메인별이다. 상세 데이터, 매트릭스, 예시는 `references/` 디렉토리에 별도 파일로 분리하고 본문에서 상대경로로 참조한다. description 프론트매터는 40~80자를 권장한다.

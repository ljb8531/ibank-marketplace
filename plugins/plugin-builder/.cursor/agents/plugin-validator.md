## 역할
최종 품질 보증 및 배포 가이드를 담당한다.

## 사용 스킬
- `validate-and-report`: 전체 검증 체크리스트 실행 및 리포트

## 절차
1. 구조 유효성 검증 (매니페스트, 네이밍, 프론트매터)
2. 글자수 준수 검증
3. 논리적 연결성 검증 (Agent→Skill 참조, T-T-D-A-V 체인)
4. MCP 연동 유효성 검증
5. 검증 리포트 출력
6. 통과 시 배포/설치 가이드 제공

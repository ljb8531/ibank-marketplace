## 목적
플러그인이 해결할 업무의 범위(scope)를 정의한다.

## 절차
1. "이 플러그인으로 어떤 업무를 자동화하려 하시나요?" 질의
2. "이 업무의 최종 산출물은 무엇인가요?" 질의
3. "이 업무를 얼마나 자주 수행하시나요?" 질의
4. 답변을 [업무명 / 산출물 / 빈도]로 정리하여 확인

## 완료 기준
- 업무명, 산출물, 빈도 3개 항목 모두 확정

## 실패 시
- 사용자가 답변이 모호하면 구체적 예시를 요청

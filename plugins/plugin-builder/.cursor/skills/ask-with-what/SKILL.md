## 목적
MCP 서버 연동 필요성과 사용자 범위를 식별한다.

## 절차
1. "이 업무에서 사용하는 외부 도구/서비스는?" 질의
2. "참조하는 문서, DB, API가 있나요?" 질의
3. "팀 내 다른 사람도 이 플러그인을 사용할 건가요?" 질의
4. 답변을 [도구명 / 연동방식 / 사용자범위]로 정리

## 완료 기준
- 외부 도구 목록 확정 (없으면 "없음"으로 확정)

## 실패 시
- 도구명이 불명확하면 URL이나 화면 공유를 요청

## 목적
모든 Skill/Agent md가 500자 미만을 유지하도록 사전 설계한다.

## 절차
1. 분류된 각 컴포넌트의 내용량을 추정
2. 500자 초과 예상 항목 식별
3. 세분화 기준 적용: T-T-D-A-V 단계별 / 분기 조건별 / 도메인별
4. 세분화 결과로 최종 파일 목록과 디렉토리 구조 확정

## 완료 기준
- 모든 예상 컴포넌트가 500자 미만 기준을 충족하는 구조

# 사용 가이드
- 커서의 프롬프트 입력창에 아래 내용과 같이 작성하여 실행 합니다.
- 전체 내용을 아래와 같은 틀에 맞추어 작성하여 입력하시길 권장합니다.

** 주의사항 **
- "# 브리프 부분" 이하에 상세 내용을 입력 합니다.
- "/orchestrator" 이 텍스트가 실제 요청을 실행하는 명령어 임으로 절대 빠지면 안됩니다.
- "/orchestrator" 이 텍스트는 가능하면 복사해서 붙여넣지 마시고 직접 타이핑 해주시길 권장 합니다.

# 예시 프롬프트
'''
# 브리프 부분
여기에 리서치를 원하는 내용이나, 고객사 정보, 요청 사항들을 정리하여 입력합니다.

---

위와 같이 OOO에서 회사 홈페이지 리뉴얼 요청을 했는데

/orchestrator 
관련해서 리서치를 좀 진행 해줄래?

'''
